What can it do?
==========

Simple wrapper around maim and xclip for doing screenshots.</br>
For help write</br>
`$ lumes -h`

Capabilities
-----------

1. Selection area
2. Move screen to clipboard (by default)
3. Save screenshot in file

Requirements
----------

1. maim
2. xclip

In future
----------

1. Save path to save screenshoots
2. Support Wayland
